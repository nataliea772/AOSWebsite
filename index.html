Autonomous Robotics System

AOS - ROS2

Our project focuses on developing an advanced robotic control system using ROS2, aimed at enhancing the functionality and autonomy of robots. By leveraging the power of the Autonomous Operating System (AOS) server, our system translates complex configuration files into executable Python code, enabling robots to perform tasks such as navigation and object manipulation seamlessly. With a modular design and the implementation of the visitor design pattern, our project ensures easy addition of new features and scalability. This transition from ROS1 to ROS2 brings improved performance, real-time capabilities, and enhanced security, making our system robust and adaptable for various robotic applications. We are excited about the potential impact of our project in advancing the field of robotics, providing a flexible, efficient, and scalable solution for autonomous robot control.
